Dear class,

A quick note regarding tools such as Git and GitHub. You do **not** need to learn very much about these technologies in order to participate fully in this course. In our case, the GitHub repository simply functions like a **folder on the Internet** that contains a number of subdirectories with course materials. All that is really required is the ability to navigate to a webpage given a URL and then browse the folder hierarchy to locate files. This is a very straightforward skill to learn, and the focus of the course will remain on the strategic concepts rather than on mastering development tools.
